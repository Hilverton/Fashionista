<h3>O que foi usado?</h3>
<ul>
	<li>React Js</li>
	<li>React Router Dom</li>
	<li>Fetch API</li>
	<li>Redux</li>
	<li>BEM CSS</li>
</ul>

<h3>Executando o projeto</h3>
<ul>
	<li>Clone o repositório</li>
	<li>Instale as dependências:</li>
	```sh
	$ npm install
	$ yarn install
	```
	<li>Execute o projeto</li>
	```sh
	$ npm start
	$ yarn start
	```
</ul>
